# WeatherApp

**WeatherApp** is a simple and user-friendly weather forecasting application built with Vanilla JavaScript. It fetches real-time weather data from an external API and displays current weather conditions and forecasts for any city around the world.

## Features

- Fetches and displays real-time weather information, including:
  - Current temperature
  - Humidity
  - Wind speed
  - Weather condition (e.g., sunny, cloudy, rainy)
- Responsive design for seamless use on any device.
- Search functionality to retrieve weather data for any city worldwide.
- Clean and simple UI for easy interaction.
